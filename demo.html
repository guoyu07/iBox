API说明

内置属性：
title 窗口标题
contentType 文档类型，默认text ,可设定为ajax,selector
showButton 是否显示按钮栏
showOk 是否显示确定按钮
showCancel 是否显示取消按钮
okBtnName 确定按钮名称
cancelBtnName 取消按钮名称
width 窗体宽
height 窗体高
timeout 自动关闭时间
draggable 是否可拖动
onok 确定按钮绑定事件
oncancel 取消按钮绑定事件
onopen 窗口打开绑定事件

内置函数：
setTitle(obj) 给窗体重新设置标题
setContent(obj) 给窗体重新设置内容
hideButton(obj) 隐藏按钮,obj为按钮名称,'ok'确定按钮,'cancel'取消按钮,'button'按钮栏
showButton(obj) 显示按钮
setButtonTitle(obj) 设置按钮名称
setWidth(obj) 设置窗体宽度
hideContent() 隐藏内容框

示例：

直接显示内容

$("#test1").click(function(){
	$.iBox.open('这是直接显示的内容', {title: 'Hello World'});
});

确定、取消的事件

$("#test3").click(function(){
	$.iBox.open('弹窗事件测试？', {
		title:'提示',
		onok:function(obj){
			obj.setContent('这是点击测试');
			obj.hideButton('ok');
		},
		oncancel:function(obj){
			alert('点击取消按钮后！');
			obj.close();//增加事件方法后需手动关闭弹窗
		}
	});
});

使用jQuery的选择器为弹窗提供内容

$("#test4").click(function(){
	$.iBox.open('#boxcontent', {title:'提示'});
});
<div style="display:none" id="boxcontent">ID方式取元素里的内容</div>


$("#test5").click(function(){
	$.iBox.open('.boxcontent', {title:'提示',contentType:'selector'});
});
<div style="display:none" class="boxcontent">Class方式元素里的内容</div>
ID方式取元素里的内容
Class方式元素里的内容

使用Ajax的方式为弹窗提供内容

$("#test6").click(function(){
	$.iBox.open('ajax.html', {title:'AJAX得到服务器上的内容', contentType:'ajax'});
});

设置弹窗的宽度和高度

$("#test7").click(function(){
	$.iBox.open('ajax.html', {title:'设置弹窗的宽度和高度', contentType:'ajax', width:600, height:300});
	//dialog默认宽度为300,高度是自适应的
});

不允许拖拽、自动关闭

$("#test8").click(function(){
	$.iBox.open('不允许拖拽、自动关闭', {
		title:'天上人间', 
		draggable:false,
		timeout:5,
		showCancel:false,
		okBtnName:'关闭'
	});
});

修改确定和取消的按钮名字

$("#test10").click(function(){
	$.iBox.open('修改确定和取消的按钮名字', {title:'修改按钮名字', okBtnName:'保存', cancelBtnName:'放弃'});
});

不显示某个按钮、按钮栏

$("#test11").click(function(){
	$.iBox.open('不显示标题和按钮栏', {
		title:'测试',
		showButton:false,//不显示按钮栏
		showOk:false,//不显示确定按钮
		showCancel:false//不显示取消按钮
	});
});
